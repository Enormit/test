Thêm Tài Khoản Mới

🔑 Hướng dẫn lấy Cookies:

  1. Đăng nhập tài khoản trên website game hoathinh3d.co
  2. Nhấn phím F12 (hoặc Click chuột phải -> chọn Kiểm Tra) -> chuyển sang tab Console.
  3. Sao chép đoạn code dưới đây dán vào Console rồi bấm Enter: copy(document.cookie)
  4. Dán chuỗi cookies vừa copy được vào ô nhập liệu bên dưới.
